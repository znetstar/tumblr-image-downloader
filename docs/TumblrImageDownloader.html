(async) scrapeBlog(options) → {Promise|Promise.<Array.<Photo>>}

Source:
  • TumblrImageDownloader.js, line 370
Iterates through all pages in a blog. By default photos are emitted via the TumblrImageDownloader#photo event and not resolved with the Promise. Set `optioons.returnPhotos` to `true` to return photos.
Example
let downloader = new TumblrImageDownloader();
downlaoder.on('photo', () => { 'do something with photo' });
downloader.scrapeBlog({ blogSubdomain: 'blah' });
